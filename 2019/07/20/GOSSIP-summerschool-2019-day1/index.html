- 攻击向量

- 计算机的储存结构以及执行指令模型：取指（）、译指、转换为多条micro指令、分发到多个执行单元。

- TLB 侧信道攻击

- 驱逐TLB表项，victim访问后TLB存在，根据translate地址时间，判断。与之前类似

- Hyper-Threading 攻击者与目标在同一个物理core，不同逻辑core 共享较多

- Same core，分时复用，共享的资源较少

- Cross core，共享的资源更少，page table、DRAM、LLC
